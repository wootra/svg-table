SVG Table

SVG Table is a powerful library for creating dynamic, customizable tables using SVG. It offers a flexible approach to table creation, allowing for complex layouts and interactive elements.

Installation

npm install @shjeon0730/svg-table

Usage

import { SvgTable } from '@shjeon0730/svg-table';

const table = new SvgTable({
  width: 500,
  height: 300,
  rows: 3,
  columns: 3
});

// Customize your table here

document.body.appendChild(table.getSvgElement());

Features

Flexible Layout

Create tables with custom layouts, including merged cells and nested structures.

SVG Power

Leverage the full capabilities of SVG for rich, interactive table designs.

Responsive Design

Build tables that adapt seamlessly to different screen sizes and devices.

Easy Integration

Seamlessly integrate with popular frameworks like React, Vue, and Angular.

Examples

Here's a basic example of creating a 3x3 table:

const table = new SvgTable({
  width: 300,
  height: 200,
  columns: 3,
  rows: 3
});

table.cells.forEach((cell, index) => {
  cell.setText(`Cell ${index + 1}`);
  cell.setStyle({
    fill: index % 2 === 0 ? '#f0f0f0' : '#e0e0e0',
    stroke: '#000000',
    'stroke-width': 1
  });
});

document.body.appendChild(table.getSvgElement());

API Reference

SvgTable

The main class for creating and managing SVG tables.

new SvgTable(options: TableOptions)

Options include:

  • width: number | string
  • height: number | string
  • columns: number
  • rows: number
  • responsive: boolean (optional)

TableCell

Represents an individual cell in the table.

cell.setText(text: string)
cell.setContent(svgContent: string)
cell.setStyle(styles: object)
