See It in Action

Installation

npm install @wootra/svg-table

How to Use


import { SvgTable } from '@wootra/svg-table';

const table = new SvgTable({
  width: 500,
  height: 300,
  rows: 3,
  columns: 3
});

// Customize your table here

document.body.appendChild(table.getSvgElement());
