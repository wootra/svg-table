SVG Table

SVG Table is a powerful and flexible library for creating dynamic, interactive tables using SVG. It provides a simple API to generate complex table structures with customizable styles and behaviors.

Features

  • Create tables using SVG for maximum flexibility and styling options
  • Support for both horizontal and vertical table layouts
  • Customizable cell contents, including text, shapes, and nested SVG elements
  • Easy-to-use API for defining table structure and styling
  • Built-in support for responsive design and mobile optimization
  • Lightweight and performant, with minimal dependencies

Installation

To install SVG Table, run the following command in your project directory: 

npm install @wootra/svg-table

Basic Usage

Here's a simple example of how to create a basic table using SVG Table:

Basic Table Example
This example creates a 3x3 table with alternating background colors. 
import { SvgTable, TableCell } from '@wootra/svg-table';

const table = new SvgTable({
  width: 300,
  height: 200,
  columns: 3,
  rows: 3
});

table.cells.forEach((cell, index) => {
  cell.setText(`Cell ${index + 1}`);
  cell.setStyle({
    fill: index % 2 === 0 ? '#f0f0f0' : '#e0e0e0',
    stroke: '#000000',
    'stroke-width': 1
  });
});

document.body.appendChild(table.getSvgElement());

Advanced Features

Custom Cell Content

SVG Table allows you to customize cell content with any valid SVG elements. Here's an example of adding a circle to a cell:

Custom Cell Content Example
import { SvgTable, TableCell } from '@wootra/svg-table';

const table = new SvgTable({
  width: 300,
  height: 100,
  columns: 3,
  rows: 1
});

const cell = table.getCell(0, 1);
cell.setContent(
  '<circle cx="50" cy="50" r="40" fill="red" />'
);

document.body.appendChild(table.getSvgElement());

Responsive Tables

SVG Table supports responsive design out of the box. You can create tables that automatically adjust to different screen sizes:

Responsive Table Example
import { SvgTable, TableCell } from '@wootra/svg-table';

const table = new SvgTable({
  width: '100%',
  height: 'auto',
  columns: 3,
  rows: 3,
  responsive: true
});

// Add content to cells...

document.body.appendChild(table.getSvgElement());

API Reference

SvgTable

The main class for creating and managing SVG tables.

new SvgTable(options: TableOptions)

Options include:

  • width: number | string
  • height: number | string
  • columns: number
  • rows: number
  • responsive: boolean (optional)

TableCell

Represents an individual cell in the table.

cell.setText(text: string)
cell.setContent(svgContent: string)
cell.setStyle(styles: object)
